下载与安装

通过 SkillHub 一键安装，或手动复制到本地，即刻开始使用 SoC Build。

方式一：SkillHub 安装 推荐

OpenClaw 用户可直接从 SkillHub 安装，一行命令搞定。

1

确保 OpenClaw 已安装

OpenClaw 是 SoC Build 的运行环境。访问 openclaw.ai 下载对应版本。

2

安装 SkillHub CLI

在终端执行以下命令安装 CLI 工具：

npm install -g skillhub-cli
3

安装 soc-build

安装完成后，一条命令即可：

skillhub install soc-build

✅ 安装成功！现在可以在 OpenClaw 中使用 SoC Build 了。

方式二：手动安装

不想用 SkillHub？也可以手动复制到本地目录。

1

克隆仓库

将 soc-build skill 仓库克隆到本地：

git clone https://github.com/your-repo/soc-build.git
2

复制到 skills 目录

将整个目录复制到 OpenClaw 的 skills 工作目录：

cp -r soc-build/ ~/.openclaw/workspace/skills/
3

验证安装

重启 OpenClaw 后，在对话中发送：

帮我初始化一个 soc 项目
💡 如果提示找不到 skill，检查 ~/.openclaw/workspace/skills/ 目录下是否存在 soc-build/ 文件夹及其 SKILL.md 文件。

环境要求

操作系统

macOS / Linux / Windows (WSL)

Python

Python 3.8+

python3 --version

依赖包

pandas, numpy

pip3 install pandas numpy

OpenClaw

v0.2.0+

openclaw --version
📦 依赖安装：安装 skill 后，如遇脚本运行报错，请确保已安装 Python 依赖：
pip3 install pandas numpy openpyxl

更新与卸载

🔄

更新 skill

通过 SkillHub 安装的用户，运行：

skillhub update soc-build
🗑

卸载

删除 skills 目录下的 soc-build 文件夹即可：

# SkillHub 用户
skillhub uninstall soc-build

# 手动安装用户
rm -rf ~/.openclaw/workspace/skills/soc-build
